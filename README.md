
DnD Style Character Creator
Note: this project was developed using PyCharm 2025.2.1.1 and can be ran by opening the project using an IDE and running Gui.py or be launching the exe file of the application.

A desktop application for generating fully randomized or customizable characters using Dungeons & Dragons 5th Edition rules.This application also allows the user to export their created character to a PDF file.

The application allows users to create characters that includes the following stats and information:

*Race and subrace

*Class and level

*Ability scores

*Equipment

*Spells

*Personality traits

*Alignment

*Background
